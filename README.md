# MYU Browser

一款为 Windows 打造的轻量级「低调浏览」工具。外表像一个普通的文档窗口，内里却是基于 WebView2 的完整现代浏览器。

它专为「需要随时藏起来」的场景设计：一键老板键隐藏到托盘、整窗透明度无级调节、窗口标题伪装、无边框极简界面，同时保持极小的资源占用和便携化的数据存储。

## 核心特性

- **隐蔽**：老板键（默认 `Alt+Q`，可自定义）一键隐藏到托盘，自动暂停网页媒体并挂起进程降内存；再次触发即刻唤醒并置顶夺焦。
- **透明**：整窗透明度 0–100% 无级调节，支持鼠标移出自动降透明。
- **伪装**：窗口标题伪装、强制深色、图片淡化、极简模式，界面无边框自绘、低调不显眼。
- **浏览**：地址栏导航、书签收藏、单窗口浏览、`Ctrl+滚轮` 缩放。
- **轻量便携**：设置与缓存随程序目录走，缓存上限可控、退出可清理；整个文件夹可直接拷贝迁移。

## 技术栈

C# · .NET 8 · Windows Forms · WebView2 · Velopack
